# ROI 测算 Excel 模板

*采购谈判风格测试 - 合作伙伴收益测算*

---

## 使用说明

1. 复制以下表格到 Excel
2. 黄色单元格为可输入项，根据客户实际情况填写
3. 灰色单元格为自动计算公式
4. 可根据需要调整假设参数

## Excel 公式参考

```excel
// 表 2 公式
B5 (预计渗透企业数) = B3 * B4
B8 (年测评总人次) = B6 * B7
B11 (年销售额) = B8 * B9
B12 (合作伙伴分成收入) = B11 * B10
B13 (我方收入) = B11 * (1-B10)
B17 (总成本) = B14 + B15 + B16
B20 (我方毛利润) = B13 - B17
B21 (毛利率 - 合作伙伴) = B12 / B11
B22 (毛利率 - 我方) = B20 / B11

// 表 4 公式
B27 (年度净收益 - 合作伙伴) = B25 - B26
B28 (ROI - 合作伙伴) = B27 / B26
B29 (投资回收期) = B26 / (B27/12)
B32 (年度净收益 - 我方) = B30 - B31
B33 (ROI - 我方) = B32 / B31
B34 (盈亏平衡点) = B31 / (B9*(1-B10) - 2)
```
