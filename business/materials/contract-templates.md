## 第二条 服务内容与标准

2.1 甲方提供的 API 服务包括:
- 测评创建接口

- 数据分析接口
- 技术支持服务

2.2 服务标准:
- API 可用性：≥ 99.9%
- 响应时间：≤ 500ms (P95)
- 技术支持：5×8 小时在线支持
- 故障响应：4 小时内响应

2.3 甲方有权根据技术发展升级 API 版本，但应保证向后兼容。

## 第四条 数据安全与保密

4.1 数据保护:
- 甲方应采取合理措施保护乙方及终端用户数据
- 数据传输采用 TLS 1.3 加密
- 数据存储采用 AES-256 加密

4.2 数据使用:
- 甲方仅可将数据用于提供服务之目的
- 未经乙方同意，甲方不得将数据用于其他用途
- 甲方不得向任何第三方披露数据

4.3 数据保留:
- 测评数据保留期限：_______年
- 保留期满后，甲方应删除相关数据
- 乙方可随时要求导出数据

4.4 保密义务:
- 双方应对合作过程中知悉的商业秘密保密
- 保密期限：合同终止后 3 年
- 违约方应赔偿守约方全部损失

## 第六条 陈述与保证

6.1 甲方保证:
- 拥有提供 API 服务的合法权利
- 服务不侵犯任何第三方知识产权
- 服务符合国家相关法律法规要求

6.2 乙方保证:
- 拥有使用 API 服务的合法权利
- 不会利用服务从事违法活动
- 不会反向工程、破解 API 服务

---

## 第七条 违约责任

7.1 任何一方违约，应赔偿守约方因此遭受的全部损失。

7.2 甲方服务不达标:
- 可用性低于 99.9%，按 SLA 赔偿
- 重大故障导致乙方损失的，赔偿直接损失

7.3 乙方违约:
- 逾期付款的，按第 3.4 条支付违约金
- 超范围使用的，甲方有权终止合作并索赔

---

## 第八条 合同变更与终止

8.1 合同变更:
- 需双方协商一致，签订书面补充协议

8.2 合同终止情形:
- 合同期满未续约
- 双方协商一致终止
- 一方严重违约，守约方单方终止
- 不可抗力导致合同无法履行

8.3 合同终止后:
- 乙方应停止使用 API 服务
- 甲方应返还或销毁乙方数据
- 保密条款继续有效

---

## 第九条 不可抗力

9.1 不可抗力包括：自然灾害、战争、政府行为、网络攻击等。

9.2 遭遇不可抗力的一方应及时通知对方，并提供证明。

9.3 不可抗力导致合同无法履行的，双方互不承担违约责任。

---
