### 3. 首轮接触话术

**文件**: `scripts/outreach-scripts.md`

**内容**:

#### Cold Email 模板 (3 个版本)
- 版本 A: 产品导向型 (适合 HR SaaS)
- 版本 B: 痛点导向型 (适合采购 SaaS)
- 版本 C: 资源互换型 (适合平台)

#### 电话销售 Script
- 开场白 (30 秒)
- 需求探询 (2 分钟)
- 价值陈述 (2 分钟)
- 异议处理 (4 种常见异议)
- 促成下一步

#### 微信跟进话术
- 首次添加好友
- 资料发送后跟进
- 未回复跟进 (第 2/5 天)
- 会议邀约
- 会议后跟进
- 长期跟进 (1/3/6 个月)

#### 常见 Q&A
- 与竞品差异
- 准确性保证
- 数据安全保障
- 价格异议处理
- 技术对接问题

**用途**: 销售团队标准化触达客户

## 🎯 使用指南

### 新客户开发流程

1. **客户筛选**: 查阅 `target-customers.md`，选择目标客户
2. **初次触达**: 使用 `outreach-scripts.md` 中的 Cold Email 模板
3. **电话跟进**: 使用 `outreach-scripts.md` 中的电话 Script
4. **演示会议**: 使用 `api-partnership-ppt-outline.md` 进行产品演示
5. **ROI 测算**: 使用 `roi-calculation-template.md` 帮助客户算账
6. **商务谈判**: 参考 `pricing-system.md` 进行价格谈判
7. **合同签署**: 使用 `contract-templates.md` 起草合同
8. **上线支持**: 参考 `bd-action-plan.md` 中的对接流程
