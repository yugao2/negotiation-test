# 采购谈判风格测试系统

一个 MBTI 风格的采购谈判风格测试 Web 应用，用户通过 28 道测试题，获得自己的谈判风格类型（16 种之一）。

## 📁 项目结构

```
negotiation-test/
├── index.html          # 入口页
├── test.html           # 测试页
├── result.html         # 结果页
├── css/
│   └── styles.css      # 全局样式
├── js/
│   ├── questions.js    # 28 道测试题目
│   ├── styles.js       # 16 种风格数据
│   ├── score.js        # 计分逻辑
│   ├── test.js         # 测试页交互
│   ├── result.js       # 结果页逻辑
│   └── poster.js       # 海报生成
├── assets/             # 资源文件（可选）
└── README.md           # 本文件
```

## 🚀 快速开始

### 方式 1：使用 Python 内置服务器

```bash
cd /root/.openclaw/workspace/negotiation-test
python3 -m http.server 8080
```

然后访问：http://localhost:8080

### 方式 2：使用 Node.js 的 http-server

```bash
npm install -g http-server
cd /root/.openclaw/workspace/negotiation-test
http-server -p 8080
```

### 方式 3：直接用浏览器打开

直接双击 `index.html` 文件即可在浏览器中打开（部分功能可能受限于本地文件协议）。

- 分享文案复制

## 🎯 测试维度

1. **决策风格（A/I）**：分析型 vs 直觉型
2. **关系导向（R/T）**：关系型 vs 任务型
3. **风险态度（C/B）**：谨慎型 vs 大胆型
4. **沟通方式（D/P）**：直接型 vs 说服型

## 📊 计分规则

- 每个维度 7 道题
- A≥4 为分析型 (A)，否则为直觉型 (I)
- R≥4 为关系型 (R)，否则为任务型 (T)
- C≥4 为谨慎型 (C)，否则为大胆型 (B)
- D≥4 为直接型 (D)，否则为说服型 (P)
