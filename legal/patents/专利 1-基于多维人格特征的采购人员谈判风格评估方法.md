# 发明专利申请书

## 发明名称
**基于多维人格特征的采购人员谈判风格评估方法**

---

## 一、技术领域

本发明属于人力资源测评技术领域，具体涉及一种基于多维人格特征的采购人员谈判风格评估方法，尤其适用于供应链采购场景中的谈判风格分析与评估。

---

## 二、背景技术

### 2.1 现有技术现状

当前采购人员能力评估主要存在以下问题:

1. **通用化测评**: 现有心理测评工具 (如 MBTI、大五人格) 为通用模型，未针对采购场景优化
2. **维度单一**: 现有评估多聚焦于沟通能力、谈判技巧等显性技能，忽视人格特质等隐性因素
3. **缺乏场景适配**: 未考虑供应链采购的特殊场景 (如 VE/VA 价值工程、供应商管理、降本增效等)
4. **评估结果应用有限**: 现有评估结果难以直接指导采购人员的岗位配置与能力提升

### 2.2 现有技术缺陷

| 现有技术 | 缺陷 |
|---------|------|
| MBTI 人格测试 | 通用模型，未针对采购场景优化 |
| 采购能力认证 | 侧重知识技能，忽视人格特质 |
| 谈判技巧培训 | 侧重方法传授，缺乏个性化评估 |
| 360 度评估 | 主观性强，缺乏科学测评工具 |

### 2.3 本发明要解决的技术问题

1. 提供专用于采购场景的人格测评模型
2. 建立采购谈判风格与人格特质的映射关系
3. 输出可操作的评估结果与改进建议

---

## 三、发明内容

### 3.1 技术方案概述

本发明提供一种基于多维人格特征的采购人员谈判风格评估方法，包括以下步骤:

**步骤 S1**: 构建采购人员人格特征测评模型
- 采用 16 种动物形象隐喻的人格分类体系
- 定义 4 个核心维度: 决策风格、沟通偏好、风险态度、关系导向
- 每个维度包含 4 个子维度，共 16 种人格类型

**步骤 S2**: 设计采购场景专用测评题库
- 包含 VE/VA 价值工程场景题目
- 包含供应商管理场景题目
- 包含降本增效场景题目
- 包含 S&OP 产供销协同场景题目

**步骤 S3**: 实施测评并计算人格特征得分
- 采用 Likert 5 点量表评分
- 计算各维度得分并确定主导人格类型
- 生成人格特征雷达图

**步骤 S4**: 映射谈判风格并输出评估报告
- 建立人格类型与谈判风格的映射关系
- 输出谈判风格画像 (如"老虎型 - 竞争主导"、"猫头鹰型 - 分析主导"等)
- 提供针对性改进建议与培训方案

|---------|---------|---------|------|------|

| 🦅 鹰 | 战略型、远见、全局思维 | 战略型谈判 | 长远规划、布局能力 | 细节关注不足 |

| 🐘 大象 | 稳健型、沉稳、可靠 | 稳健型谈判 | 风险低、可信赖 | 创新不足 |
| 🦉 猫头鹰 | 分析型、严谨、逻辑 | 分析型谈判 | 数据驱动、精准 | 决策慢 |

### 3.3 测评维度设计

**维度 1: 决策风格**
- 子维度: 果断性、分析深度、风险偏好、时间压力应对

**维度 2: 沟通偏好**
- 子维度: 表达方式、倾听能力、反馈风格、冲突处理

**维度 3: 关系导向**
- 子维度: 关系重视度、信任建立、长期导向、合作意愿

**维度 4: 任务导向**
- 子维度: 目标坚持、结果导向、效率优先、质量标准

### 3.4 谈判风格映射规则

```
IF 决策风格 = 果断 AND 任务导向 = 高 THEN 谈判风格 = "竞争型"
IF 决策风格 = 分析 AND 关系导向 = 中 THEN 谈判风格 = "分析型"
IF 沟通偏好 = 友善 AND 关系导向 = 高 THEN 谈判风格 = "关系型"
IF 决策风格 = 灵活 AND 沟通偏好 = 灵活 THEN 谈判风格 = "灵活型"
...
```

### 3.5 评估报告输出

评估报告包含:
1. 人格类型判定 (主类型 + 副类型)
2. 谈判风格画像
3. 各维度得分雷达图
4. 优势与劣势分析
5. 针对性改进建议
6. 推荐培训方案
7. 适合采购场景建议

## 五、具体实施方式

### 实施例 1: 在线测评系统实施

1. **系统部署**: 基于 Web 的在线测评平台
2. **题库规模**: 60-80 道测评题目
3. **测评时长**: 15-20 分钟
4. **评分算法**: 维度得分 = Σ(题目得分×权重)
5. **类型判定**: 最高得分维度确定主类型

### 实施例 2: 企业内训场景实施

1. **线下测评**: 纸质问卷或平板测评
2. **工作坊形式**: 测评 + 解读 + 培训一体化
3. **团队分析**: 汇总团队人格分布
4. **配置建议**: 基于团队风格互补提出配置建议

### 实施例 3: AI 动态评估实施

1. **数据采集**: 历史谈判记录、沟通记录、决策记录
2. **AI 分析**: 基于 NLP 的行为模式分析
3. **动态更新**: 定期更新评估结果
4. **预测功能**: 预测不同场景下的谈判表现
