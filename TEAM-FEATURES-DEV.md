## 🔧 核心函数

| 函数名 | 功能 | 参数 |
|--------|------|------|
| `goToTeam()` | 跳转到团队页面 | 无 |
| `switchTeamTab(tab)` | 切换团队 Tab | 'create'/'dashboard'/'compare'/'report' |
| `createTeam()` | 创建团队 | 读取表单输入 |
| `copyInviteCode()` | 复制邀请码 | 无 |
| `viewTeamDashboard()` | 查看仪表盘 | 无 |
| `updateTeamDashboard()` | 更新仪表盘数据 | 无 |
| `compareMember(name)` | 对比成员 | 成员姓名 |
| `runComparison()` | 执行对比 | 读取选择框 |
| `exportTeamReport()` | 导出报告 | 无 |
